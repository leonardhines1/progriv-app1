   🔨  ЯК ЗІБРАТИ «ПРОГРІВ.EXE» НА WINDOWS
═══════════════════════════════════════════════════

Ця інструкція для ТЕБЕ (адміна), а не для фармерів.
Тобі потрібен будь-який комп'ютер з Windows 10/11.


КРОК 1 — Встановити Python (якщо ще немає)
──────────────────────────────────────────
1. Відкрий https://www.python.org/downloads/
2. Натисни «Download Python 3.x.x»
3. Запусти інсталятор
   ⚠️  ОБОВ'ЯЗКОВО постав галочку:
   [✓] Add Python to PATH    ← це найважливіше!
4. Натисни «Install Now»
5. Дочекайся завершення


КРОК 2 — Скопіювати цю папку на Windows
────────────────────────────────────────
Скопіюй ВСЮ папку «Програма Windows» на Windows-комп.
Наприклад, на Робочий стіл.

Структура папки:
  📁 Програма Windows/
  ├── main.py
  ├── requirements.txt
  ├── Progriv_win.spec
  ├── ЗБІРКА.bat
  ├── 📁 app/
  │   ├── __init__.py
  │   ├── api_client.py
  │   ├── constants.py
  │   ├── error_parser.py
  │   ├── generator.py
  │   ├── gui.py
  │   └── key_codec.py
  └── 📁 assets/
      ├── icon.png
      └── icon.ico


КРОК 3 — Зібрати .exe
─────────────────────
1. Відкрий папку «Програма Windows»
2. Двічі клікни на «ЗБІРКА.bat»
3. З'явиться чорне вікно — чекай 1-3 хвилини
4. Коли побачиш «✅ ГОТОВО!» — відкриється папка dist/
5. Там лежить файл «Прогрів.exe» (~30-40 MB)


КРОК 4 — Перевірити
───────────────────
1. Двічі клікни на «Прогрів.exe»
2. Windows SmartScreen може показати попередження:
   → Натисни «Докладніше» (More info)
   → Натисни «Усе одно запустити» (Run anyway)
3. Програма має запуститися — вікно з вкладками


КРОК 5 — Роздати фармерам
─────────────────────────
Фармерам потрібен ТІЛЬКИ один файл:
  → dist/Прогрів.exe

Надішли їм цей файл + ІНСТРУКЦІЯ_ФАРМЕРУ.txt

   ❓ ПРОБЛЕМИ
═══════════════════════════════════════════════════

❓ «Python не знайдено»
   → Встанови Python (Крок 1), обов'язково з галочкою PATH

❓ Помилка при збірці
   → Закрий антивірус, спробуй ще раз

❓ .exe не запускається
   → Спробуй запустити від імені адміністратора
